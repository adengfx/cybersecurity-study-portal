Offensive Security Intro

Hack your first website legally in a safe environment and experience an ethical hacker's job.

Task 1 — What is Offensive Security?
  • The core principle of offensive security is "to outsmart a hacker, you need to think like one."
  • Offensive security involves breaking into computer systems, exploiting software bugs, and finding loopholes in applications — all with authorisation. The goal is to understand attacker tactics so that system defences can be strengthened.
  • It is the counterpart to defensive security: where defenders protect systems, offensive practitioners probe them to find weaknesses before real attackers do.

Task 2 — Hacking Your First Machine
  • TryHackMe uses lab machines to create simulated environments as practical complements to lessons. This task used a fake bank application called FakeBank.
  • Gobuster is a command-line tool used to brute-force hidden directories and pages on a website. It takes a wordlist and tries each entry against the target — reporting any pages that exist.
  • Key Gobuster flags: -u specifies the target URL, -w specifies the wordlist to iterate through.
  • Running Gobuster against FakeBank revealed a hidden page: /bank-transfer. Navigating to this page and transferring $2,000 from account 2276 to 8881 completed the task.

Steps: Open terminal → Run Gobuster with -u and -w flags → Discover /bank-transfer → Transfer funds between accounts → Flag appears on account page.

Task 3 — Careers in Cyber Security
  • A career in cyber security is achievable through consistent learning and hands-on practice — building skills incrementally over time.
  • Penetration Tester: Responsible for testing technology products for exploitable security vulnerabilities.
  • Red Teamer: Plays the role of an adversary, attacking an organisation and providing feedback from an enemy's perspective.
  • Security Engineer: Designs, monitors, and maintains security controls, networks, and systems to help prevent cyberattacks.
